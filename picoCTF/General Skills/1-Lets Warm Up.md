
# Descripcion 
- Si te dijera que una palabra comienza con 0x70 en hexadecimal, ¿con qué comenzaría en ASCII?

# Solución 
# Solución 1
- Utilizando Cyberchef 
- https://gchq.github.io/CyberChef/#recipe=From_Hex('0x')&input=MHg3MA
# Solución 2
- Utilizando Python
```
>>> int(0x70)
112
>>> chr(112)
'p'
>>> 
```
